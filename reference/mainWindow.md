<h1>Main Window</h1>
<p>The Main Window is where you configure and run your show. </p>
<h2>Dashboard</h2>
<p class="hcp2">Inside the Main Window is the Dashboard. 
 </p>
<p class="hcp3"><img alt="" src="../../images/MainDashboard.png" width="587" height="209" border="0" class="hcp4"></p>
<p class="hcp2">When you have installed Screen Monkey and 
 run it for the first time, the dashboard contains nothing but blank squares. 
 The squares are called &quot;slots&quot;. You will then add different 
 media elements to each of the slots. Once added, these elements are called 
 clips. As you add clips the slots should present thumbnail images representing 
 what the clip contains. </p>
<p class="hcp2">The Dashboard may be sized in any configuration 
 that appeals to you. Screen Monkey will respect your arrangement and save 
 it when you close the application. When you close and restart Screen Monkey, 
 your arrangement is reloaded.</p>
<p>You may also change the size of the thumbnail images of the clips presented 
 in the dashboard. This is controlled by visiting the <a href="../../reference/setup/Settings/Settings.md">Screen 
 Monkey Settings</a>.</p>
<p class="hcp3"><img alt="" src="../../images/MainDashboard3.png" border="0" class="hcp4"></p>
<p>The first clip in the dashboard is always the <span class="rvts12" style="font-weight: bold;">Clear 
 Layer</span> clip.</p>
<p class="hcp3"><img alt="" src="../../images/MainDashboardClearLayer.png" border="0" class="hcp4"></p>
<p>As you present a show using Screen Monkey, content is presented using 
 up to four different <a href="../WorkingWithShows/DisplayLayers.md">layers</a>. 
 Think of layers as being like a physical desktop. The desktop is similar 
 to what you see on the second monitor. If you present something on layer 
 1, it covers the desktop beneath it. And if something is presented on 
 layer 2, it will cover the desktop as well as covering whatever layer 
 1 is presenting. Perhaps you are presenting things on all four of the 
 layers and wish to stop presenting whatever is being shown on layer 2. 
 The Clear Layer clip allows you to clear whatever layer you have selected.</p>
<p>&#160;</p>
<h2>Pages</h2>
<p>The grouping of clips and slots inside the Dashboard is referred to 
 as a <span class="hcp5">Page</span>. For example, in the 
 Dashboard shown below you see that Page 1 is capable of storing eight 
 clips. If the window were sized wider or taller, more clips might be visible 
 on the page. In this configuration each page will hold eight clips. If 
 the window were sized taller so that two rows appeared, each page would 
 contain 16 clips.</p>
<p class="hcp6"><span class="rvts10"><img alt="" src="../../images/MainDashboardPage1.png" border="0" class="hcp4"></span></p>
<p>Screen Monkey places no restriction on the number of pages that may 
 store clips. There are multiple ways to change the pages so you may view 
 different pages in the Main Dashboard. </p>
<p>The first way is to click the numbers in the <span class="hcp5">Page 
 Select Bar</span> to display that particular page of clips.</p>
<p class="hcp6"><img alt="" src="../../images/MainDashboardPageSelectBar.png" border="0" class="hcp4"></p>
<p>There are four other ways of changing pages. </p>
<ol type="1">
	<li><p>Clicking the <span class="hcp5">Plus</span> or 
	 <span class="hcp5">Minus</span> symbols to the immediate 
	 right of the <span class="hcp5">Page Select</span> numbers 
	 allows you to present the next higher or next lower page.</p></li>
	<li><p>Pressing the <span class="hcp5">Page Up</span> 
	 or <span class="hcp5">Page Down</span> keys on your 
	 keyboard will also allow you to display the next higher or lower number 
	 in the page count.</p></li>
	<li><p>Each page may be assigned a <span class="hcp5">Bookmark</span> 
	 to more easily identify its use. <a href="OrganizingClips.md#Bookmarking_Pages">Click 
	 here</a> for more information on using Bookmarks.</p></li>
	<li><p>Clicking the yellow page number in the Page Select Bar will 
	 open the <span class="hcp5">Goto Page</span> dialog 
	 where you may type a page number into the field and click <span class="hcp5">OK</span> 
	 or pres <span class="hcp5">Enter</span> to display that 
	 page.<br>
	<br>
	<img alt="" src="../../images/GotoPage.png" border="0" class="hcp4"></p></li>
</ol>

<h2>Layers</h2>
<p>On the left side of the Dashboard is the <span class="hcp5">Layers</span> 
 area. This area provides an indication of the selected layer as well as 
 allowing you to select a layer by clicking on it.</p>
<p>If you wish to clear a layer you first select it here then click the 
 <span class="hcp5">Clear Layer</span> clip.</p>
<p><span class="rvts10">From the <a href="../WorkingWithShows/LayersDashboard.md">Layers 
 Dashboard</a> you are able to fade layers in and out. As the layer opacity 
 reduces, the layer buttons will change to grey to indicate the layer is 
 no longer visible.</span></p>

<h2><span class="rvts11">Info Bar</span></h2>
<p><span class="rvts10">The</span> <span class="rvts17">Info Bar</span> 
 <span class="rvts10">displays information about a clip when the mouse 
 passes over the clip. When not showing clip information it displays information 
 about the live clip on the current layer. You may use this to see information 
 such as the full clip name, duration, slides, link and effect. Screen 
 Monkey may be <a href="../../reference/setup/Settings/Settings.md">configured</a> 
 to hide the info bar if desired.</span></p>
<p class="hcp3"><img alt="" src="../../images/img_288.jpg" border="0" class="hcp4"></p>
<p>&#160;</p>
<p style="font-weight: bold;">Docking Windows</p>
<p>The windows within Screen Monkey are all docakable. This offers an extremely 
 powerful and flexible way of arranging all the different elements of the 
 software. You can arrange all the elements in the way that best suits 
 you.</p>
<p class="rvps2" style="margin-left: 24px;"><img alt="" src="../../images/img_186.jpg" width="556" height="359" border="0" class="hcp4"></p>
<p>To change the layout, drag the window to the new position. As you drag 
 the window you will see guides appear indicating where you can drop the 
 window to dock it. Dragging the window away from the main dashboard will 
 cause it to float.</p>
<p class="rvps2" style="margin-left: 24px;"><img alt="" src="../../images/img_187.jpg" border="0" class="hcp4"></p>
<p>If you drag the window outside all the other windows, it will float. 
 Dropping the window in the center of the guide will cause it to become 
 tabbed.</p>
<p class="rvps2" style="margin-left: 24px;"><img alt="" src="../../images/img_188.jpg" border="0" class="hcp4"></p>
<p>To view a window that is tabbed, click the tab to make it visible. You 
 may also pin a window so that it is initially hidden but slides out when 
 you click on the window icon. To pin a window, click the <span class="hcp5">Pin</span> 
 icon.</p>
<p class="rvps2" style="margin-left: 24px;"><img alt="" src="../../images/img_189.jpg" border="0" class="hcp4"></p>
<p>Please note that the window must be docked and not tabbed to see this 
 icon. When the window is pinned it will appear as a small button against 
 the edge it has been pinned to.</p>
<p class="rvps2" style="margin-left: 24px;"><img alt="" src="../../images/img_190.jpg" border="0" class="hcp4"></p>
<p>You now have a great deal of flexibility to layout the different parts 
 of Screen Monkey, where you want them.</p>
